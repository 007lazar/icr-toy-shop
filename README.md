## Project Overview
ICR Toy Shop is a full-stack web application for browsing and purchasing toys. It includes basic e-commerce features and is being extended with an integrated AI chatbot for conversational product search and order assistance.

### Features
- User authentication (LocalStorage)
- Product listing & filtering
- Checkout & order creation
- (In Progress) AI chatbot for natural language interaction

- ### Technology Stack
Angular • TypeScript • Node.js • Rasabot • Anaconda • Git
